# 快速上手

  1. 使用本框架前，建议在 CSS 中开启 border-box，如：

    *, *::before, *::after {
  box-sizing: border-box;
}

  2. 引入组件

    import { Button } from 'gulu-vue2'

  3. 注册组件

    export default {
  name: 'ComponentName',
  components: {
    'g-button': Button,
  },
}
